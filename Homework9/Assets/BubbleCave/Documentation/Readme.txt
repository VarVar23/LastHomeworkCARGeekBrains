Intro

Bubble Cave is a unique 3 match game template that anyone can enjoy! Different bubbles are falling down from the top. You have to match at 3 bubbles of same color by rotating the cave center. Watch for special bubbles which can make the difference during gameplay. Destroy bubbles before they reach cave border.

Template Features:

- Addictive gameplay: fast paced 3 match gameplay
- Amazing visual effects: particles and animations
- Different Bubbles: 6 bubbles of different color
- Special Bubbles: 9 special bubbles with different animations and abilities
- Score system: for leaderboards, achievements
- Optimised for mobile and tablet devices: scale on different screen sizes
- Works on all platforms: PC, MacOS, iOS, Android, WebGL

Files structure:

1) Animations: animations which are used in project
2) Fonts: main project fonts
3) Materials: all physics materials
4) Particles: all particles components used in project
5) Prefabs: object prefabs used in project
6) Scenes: scenes used in project
7) Scripts: script files
8) Sounds: sound files used in project
9) Sprites: sprite files used in project
